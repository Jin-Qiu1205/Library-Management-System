# Library Management System (React + Node.js)

This is a simple full-stack library management project built with a React frontend and a Node.js backend.

Users can:
- Add book information using a form
- View the current list of books
- See updates immediately after submission

This project was created as part of a JavaScript / web development course.

---

## Tech Stack

**Frontend**
- React
- Vite
- react-hook-form
- Fetch API
- CSS

**Backend**
- Node.js
- Express
- REST API

---
